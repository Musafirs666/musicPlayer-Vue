# musicPlayer-Vue3

## Project Setup

```sh
npm install
create .env file in root folder
create VITE_APP_FIREBASE_API_KEY = your-own-firebase-api-key
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
